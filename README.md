# Frontend Desktop

Next.js + TypeScript desktop app for:
- Site Manager
- Affected Citizen

## Run

```
cd frontend_desktop
npm install
npm run dev
```

## Open

- `http://localhost:3000`
- Main login: `http://localhost:3000/login`
- Citizen routes: `http://localhost:3000/citizen/auth`

## Notes

- Main app files are in `app/`
