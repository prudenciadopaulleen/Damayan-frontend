# Damayan Frontend

Next.js + TypeScript web app for:
- Admin
- Dispatcher
- Site Manager
- Affected Citizen

## Requirements

- Node.js 18 or newer
- npm

## How To Run

From the project root:

```bash
npm install
npm run dev
```

## Open In Browser

- App: `http://localhost:3000`
- Role selector: `http://localhost:3000/loginportal`
- Citizen login: `http://localhost:3000/citizen/login`
- Citizen signup: `http://localhost:3000/citizen/signup`
- Admin login: `http://localhost:3000/admin/login`
- Dispatcher login: `http://localhost:3000/dispatcher/login`
- Site Manager login: `http://localhost:3000/site-manager/login`

## Project Notes

- Main routes and pages are inside `app/`
- This project is a web app built with Next.js, not a React Native mobile app
